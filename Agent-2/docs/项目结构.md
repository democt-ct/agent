## 3. src/ — Cloudflare Worker（TypeScript 主 API）

### 3.1 API 路由一览

所有路由在 [src/index.ts](Agent-2/src/index.ts) 中注册：

| 方法 | 路由 | 说明 |
| --- | --- | --- |
| `GET` | `/` | 健康检查 |
| `GET` | `/map/config` | 获取高德地图公钥 |
| `POST` | `/map/search` | POI 搜索（关键词 + 城市 + 范围） |
| `POST` | `/sessions` | 创建会话 |
| `GET` | `/sessions/:id` | 获取会话详情 |
| `POST` | `/sessions/:id/requirements` | 创建出行需求（含 LLM 解释） |
| `POST` | `/sessions/:id/requirements/interpret` | 仅解释需求（不持久化） |
| `GET` | `/sessions/:id/requirements/latest` | 获取最新需求 |
| `POST` | `/sessions/:id/itineraries` | 生成行程（template / llm / agent） |
| `GET` | `/sessions/:id/itineraries/latest` | 获取最新行程 |
| `GET` | `/sessions/:id/itineraries/latest/route-plan` | 获取路线规划数据 |
| `POST` | `/sessions/:id/replan` | 重规划行程 |
| `POST` | `/sessions/:id/chat` | ★ 核心 Chat 端点：对话式交互 |
| `POST` | `/sessions/:id/messages` | 保存消息 |
| `GET` | `/sessions/:id/messages` | 获取会话消息列表 |

## 11. 核心数据流

```
用户聊天输入
    ↓
FastAPI (/api/chat)                    ← fastapi/core.py
    ↓
Cloudflare Worker (/chat)              ← src/services/orchestration/tripOrchestrator.ts
    ├─ 需求解释 (LLM)                  ← src/services/llm/requirementInterpreter.ts
    ├─ 候选池构建                      ← src/services/candidates/candidatePoolBuilder.ts
    ├─ MCP 工具调用 (高德+Web搜索)     ← src/services/mcp/mcpClient.ts
    ├─ 旅行规划流水线                  ← src/services/planning/travelPlanningPipeline.ts
    └─ 存储 (D1)                       ← src/repositories/*
    ↓
结构化行程 JSON
    ↓
FastAPI 前端渲染                       ← fastapi/static/index.html
    ├─ 高德地图落点                    ← fastapi/static/js/map.js
    ├─ 路线规划展示                    ← 高德 JS API
    └─ 行程编辑交互                    ← fastapi/static/js/app.js
```

## 12. 技术栈

| 层 | 技术 |
| --- | --- |
| **API 网关** | Cloudflare Workers（TypeScript） |
| **数据库** | Cloudflare D1（SQLite） |
| **地图服务** | FastAPI（Python）+ 高德 Web API |
| **LLM** | OpenAI Responses API（GPT-5.2） |
| **前端** | 原生 HTML/JS + 高德 JS API 2.0 |
| **爬虫** | Python + Playwright + asyncio |
| **部署** | Cloudflare Wrangler / Uvicorn |
