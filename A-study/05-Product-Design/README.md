## 两个核心认知

```
① Agent 不是所有问题的答案
   很多场景用 Workflow 更稳定、更便宜、更快。
   很多场景根本不需要 LLM。
   你的第一反应不应该是"加个 Agent"。

② 好的 Agent 产品设计 ＞ 好的 Agent 技术
   用户不在乎你用了 ToT 还是 CoT。
   用户只在乎：它能不能准确理解我、可靠完成任务、出错时优雅降级。
```
