# 安全策略

此仓库是 Prompthon Agentic Labs 及其已跟踪示例项目的存放地。主要与安全相关的面向包括已跟踪的示例代码、工作流自动化，以及任何会影响贡献如何被审查和合并的仓库配置。

## 受支持的面向

| 面向 | 支持 |
| --- | --- |
| 当前 `main` 分支内容 | 是 |
| `main` 上 lane-local `examples/` 文件夹下的示例项目 | 是 |
| 历史提交、已废弃分支或 fork | 否 |
| `references/` 下的导入材料 | 否 |

## 报告漏洞

如果此仓库已启用 GitHub 私有漏洞报告，请使用仓库的 **Report a vulnerability** 流程。

如果私有漏洞报告尚未启用：

1. 不要公开创建 issue。
2. 使用 `SUPPORT.md` 中的 Prompthon IO 社区联系路径来请求私密报告渠道。
3. 在可用私密渠道之前，仅共享高层级描述。

请包含以下内容：

- 受影响的路径或工作流
- 影响
- 清晰的复现步骤
- 验证报告所需的任何概念验证细节

## 处理预期

本仓库以尽力维护的方式进行维护。影响活跃示例代码、工作流安全性或仓库治理的报告，将优先于历史材料或仅供参考的材料。
