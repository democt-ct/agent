## 概览

Prompthon Agentic Labs 发布 Agent Systems Handbook by Prompthon：这是一份面向 AI 原生学生、实践者和构建者的实战指南，帮助他们从不同角度探索现代智能体系统。

基于 **学习、提问和创新** 构建，这个 lab 由学习者塑造，并扎根于真实的行业实践。它帮助读者理解这一领域、有效应用 AI，或通过并行路径而非单一路径构建真实系统。

## 为什么这个 lab 适合 AI 原生学习者、实践者和构建者

### 基于学习、提问和创新

这个仓库鼓励主动学习、批判性思维和实验，而不是被动消费内容。

### 由学习者构建，而不只是为学习者而建

许多贡献者本身也是学习者。这使内容始终贴近学生、新毕业生以及下一代 AI 原生构建者真正关心的问题、习惯和学习路径。

### 以真实行业实践为指导

通过 Prompthon 项目和面向行业的指导，这个 lab 与前沿团队在真实场景中如何思考、构建、迭代和评估保持一致。

### 天然面向 AI 原生

内容通过一种 AI 原生工作流创建，将 AI 辅助起草、归纳、迭代和精炼与专家指导和审阅相结合。

### 为不同路径设计，而非单一路径

这个 lab 面向不同类型的学习者和不同目标进行组织。有些人想要广泛理解和趋势感知。有些人想将 AI 工具应用到日常工作和学习中。有些人想构建真实系统和应用。这个仓库支持这三种路径，而不强制任何单一顺序。

## 这本手册涵盖什么

- AI 智能体基础和 agent-system 心智模型
- Agentic 工作流、规划、反思、工具使用和 function calling
- 智能体记忆、检索、上下文工程和 agentic RAG
- MCP、A2A、协议互操作和智能体通信边界
- LangGraph、智能体框架、托管式构建器和低代码平台
- 多智能体编排、评估、可观测性、可靠性和安全
- Deep research 智能体、客户支持智能体、source project 和 starter 示例

## 贡献者指南

如果你想为 Prompthon Agentic Labs 贡献内容，请从 contributor 文档开始，而不是使用临时性的内部工作材料。

本仓库中的公开贡献目前包括以下路径：

- `foundations/`、`patterns/`、`systems/`、`ecosystem/` 或
  `case-studies/` 中的 lab 文章
- [`radar/`](./radar/) 中的 radar 笔记
- lane-local `examples/` 文件夹中的源项目
- [`skills/`](./skills/index.mdx) 中的 practitioner skill package
- [`contributor-kit/reference-notes/`](./contributor-kit/reference-notes/README.md) 中的精选参考笔记
- 一旦某个 lab 页面准备好作为对外文章或分发载体，可放入 [`publications/`](./publications/README.md) 中的发布扩展

请从 [贡献指南](./CONTRIBUTING.md) 和 [贡献者工具包](./contributor-kit/index.mdx) 开始。这些页面定义了适用于本仓库中 lab 文章、笔记和代码的公开工作流、模板、审阅标准和放置规则。
