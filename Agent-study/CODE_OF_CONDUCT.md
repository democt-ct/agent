# 行为准则

Prompthon Agentic Labs 是一个公开的工作实验室。我们希望贡献、
批评和评审始终具体、尊重且对下一个
人有帮助。

本行为准则适用于本仓库以及用于协调实验室工作的项目关联社区
空间。

## 期望的行为

- 批评想法、结构、证据和实现细节，而不是
  人。
- 先假设对方是善意的，然后明确指出问题和支撑
  证据。
- 在讨论外部材料时，保持归属边界清晰。
- 为评审者提供足够上下文，以便无需猜测即可评估变更。
- 尊重关于范围、放置位置和仓库适配性的编辑决策。

## 不可接受的行为

- 骚扰、歧视、威胁、恐吓或人身攻击。
- 在决定已记录后，仍反复进行恶意争论、带偏线程或组织围攻。
- 未经许可发布私人信息。
- 将复制来的上游文字、截图、图表或大量代码内容冒充为仓库原生工作。
- 垃圾信息、无关推广或在项目空间内进行非项目相关招聘。

## 举报

请不要为行为问题公开创建 issue。

如果你经历或目睹了行为问题，请私下联系维护者。
如果没有明显的直接维护者联系方式，请使用
`SUPPORT.md` 中描述的路径，申请一个私下的审核渠道。

## 执行

维护者可在需要时编辑、隐藏、锁定或移除内容；要求更改；限制
参与；或将账号从项目空间中封禁，以保护
社区并保持协作可行。
