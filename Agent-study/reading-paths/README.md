# 阅读路径

这个目录将仓库从单一章节流转变为以路径为主的实验室。

## 主指南

- [探索者](./explorer.mdx)：适合想要高层次理解，并希望获得一组推荐页面而不是严格顺序的人。
- [实践者](./practitioner.mdx)：适合想在实际工作中使用智能体工具和技能，而不成为专业智能体构建者的人。
- [构建者](./builder.mdx)：适合希望沿着线性路径深入专业智能体设计和实现的工程师。
- [贡献者](./contributor.mdx)：适合想要编辑、策划或扩展仓库本身的人。

## 共享支持指南

- [Environment Setup](./environment-setup.mdx)：为 practitioner、builder 和 contributor 路径提供共享的本地环境设置和入门项目验证。
- [入门项目](./sample-projects.mdx)：当前仓库自有入门项目的公开索引。
