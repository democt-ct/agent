## RAG 检索流程（子流程）

```
用户 query: "年终奖什么时候发"
    │
    ▼
[查询改写] src/rag/query_rewriter.py
    │  LLM 将口语转为书面语: "年终奖发放规则 发放时间"
    │  模型：跟随 LLM_MODEL 环境变量（与 Agent 共用）
    │  temperature=0.1, max_tokens=100
    │  改写失败 → 静默回退到原始 query
    ▼
[向量检索] src/rag/retriever.py → search_vector()
    │  ChromaDB 余弦相似度查询
    │  模型：BAAI/bge-large-zh-v1.5，768 维
    │  距离→相似度公式: 1/(1+L2_distance)
    ▼
[BM25 检索] src/rag/retriever.py → search_bm25()
    │  jieba 分词 + rank_bm25 关键词匹配
    │  BM25 结果权重是向量的 1.5x (bm25_weight=1.5)
    ▼
[RRF 融合] src/rag/retriever.py → rrf_fusion()
    │  Reciprocal Rank Fusion: score = Σ 1/(k + rank)
    │  默认 k=10
    ▼
[重排序] src/rag/reranker.py (可选，RERANK_ENABLED=false 默认关闭)
    │  Cross-Encoder (BAAI/bge-reranker-base) 逐对评分
    │  或 Cohere Rerank API
    │  从 20 个候选中精选 3 个
    ▼
[注入 Agent] src/agents/base_agent.py → _build_system_prompt()
    │  检索结果以 "【知识库参考材料】" 区块注入 system prompt
    │  query 模式检索 top_k=3，action 模式 top_k=10
    ▼
[LLM 回答] ReAct 循环，工具调用后生成最终回答
```

## 两条执行路径对比

| 维度 | **普通路径** (绝大多数请求) | **Workflow 路径** (PLANNER_ENABLED=true) |
|------|---------------------------|------------------------------------------|
| 入口 | `chat_service.chat()` | `chat_service._execute_plan()` |
| 路由 | Orchestrator（规则+LLM） | 不路由，Planner 已指定 domain |
| Agent | 1 个 primary（+ optional secondary） | DAG 多个节点串行/并行 |
| 执行器 | `BaseAgent.run()` → ReAct 循环 | `WorkflowEngine.run()` → 拓扑排序 → 逐节点调 Agent |
| 节点类型 | — | `retrieval` / `tool` / `memory` / `review` |
| 依赖 | — | `depends_on` 列表，Kahn 拓扑排序 |

## 各模块职责速查

| 模块 | 文件 | 一句话 |
|------|------|--------|
| **RAG 管线** | `src/rag/*.py` | 文档加载→结构感知分块→向量化→FAQ+混合检索→可选重排 |
| **Embedding** | `src/rag/embedder.py` | BGE-large-zh-v1.5，ModelScope 优先下载，1024 维 |

| **BaseAgent** | `src/agents/base_agent.py` | ReAct 循环 + 30s 超时 + 工具参数校验 |
| **Planner** | `src/agents/planner.py` | 多步骤任务 DAG 拆解（关键词快速路径 + LLM 分解） |

| **WorkflowEngine** | `src/workflow/engine.py` | DAG 拓扑执行，节点重试/失败降级 |
| **ToolAgent** | `src/agents/tool_agent.py` | 工具执行 + 权限校验 + 审计日志 |
| **ReviewAgent** | `src/agents/review_agent.py` | 答案审查：事实核对 + 合规检查 |
| **Memory** | `src/memory/` | L2 对话存储 + L3 长期记忆 + 窗口压缩 |
| **Chat Service** | `src/api/service/chat_service.py` | 总编排：Planner→路由→Agent→交割→Review→Memory |
| **LLM Client** | `src/llm_client.py` | OpenAI 兼容客户端工厂，带指数退避重试 |
| **Skills - 工单生命周期** | `src/skills/ticket_lifecycle.py` | `[v2]` 创建→处理→完成→关闭→重开→升级 + FSM + 幂等 |

| **安全 - 密码** | `src/gateway/crypto.py` | `[v2]` pbkdf2_hmac 哈希 (60万轮) |

## 可配置参数速查

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `LLM_MODEL` | `deepseek-v4-flash` | 主 LLM（路由/改写/Agent 共用） |
| `LLM_BASE_URL` | `https://api.deepseek.com` | LLM API 地址 |
| `PLANNER_ENABLED` | `false` | 是否启用 DAG 任务规划 |
| `MEMORY_ENABLED` | `false` | 是否启用会话记忆 |
| `RERANK_ENABLED` | `false` | 是否启用 RAG 重排序 |
| `chunk_size` | `500` | 文档分块大小（token） |
| `chunk_overlap` | `80` | 分块重叠（token） |
| `top_k_retrieval` | `10` | RAG 检索返回条数 |
| `top_k_rerank` | `3` | 重排序后精选条数 |
| `rag_cache_ttl_seconds` | `60` | RAG 查询缓存 TTL |
