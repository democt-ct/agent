# 员工纪律处分条例

**版本：** v1.0 | **生效日期：** 2026年6月1日 | **归口部门：** 法务部
**会签部门：** 人力资源部 | **批准人：** CEO

---

## 1. 总则

为维护公司正常经营秩序、规范员工行为、保障公平公正的工作环境，特制定本条例。本条例适用于公司全体员工。纪律处分遵循"事实清楚、证据确凿、程序合法、处分适当"的原则。

---

## 2. 处分种类

按违纪情节轻重，处分分为五级：

| 级别 | 处分 | 影响 |
|------|------|------|
| 一级 | 口头警告 | 记录在案，影响当月绩效 |
| 二级 | 书面警告 | 记入个人档案，影响季度绩效与评优 |
| 三级 | 记过 | 记入档案，取消当年评优、调薪资格 |
| 四级 | 降级/降职 | 调整岗位职级与薪资 |
| 五级 | 解除劳动合同 | 严重违纪，依法解除 |

> 一年内累计两次书面警告升级为记过；累计两次记过可升级为降级或解除。

---

## 3. 违纪行为与对应处分

### 3.1 考勤纪律

| 行为 | 处分 |
|------|------|
| 月迟到/早退累计超规定次数 | 口头警告 + 扣款（见《考勤政策》） |
| 旷工1天 | 书面警告 |
| 连续旷工3天或年累计旷工7天 | 解除劳动合同 |
| 代打卡/虚假考勤 | 记过，双方均处分 |

### 3.2 工作纪律

| 行为 | 处分 |
|------|------|
| 工作时间处理私事、影响工作 | 口头警告 |
| 不服从合理工作安排 | 书面警告 |
| 失职造成公司损失（轻微） | 记过 + 赔偿 |
| 严重失职、营私舞弊造成重大损失 | 解除劳动合同 |

### 3.3 廉洁纪律

| 行为 | 处分 |
|------|------|
| 收受供应商/客户礼品未申报（价值＞200元） | 记过 + 上交 |
| 收受回扣、佣金 | 解除劳动合同，追缴违法所得 |
| 利用职务为亲属谋利、利益输送 | 解除劳动合同 |

### 3.4 信息安全与保密纪律

| 行为 | 处分 |
|------|------|
| 违规使用个人设备/网盘存储公司数据 | 书面警告 |
| 泄露一般内部信息 | 记过 |
| 泄露商业秘密、核心数据 | 解除劳动合同 + 追究法律责任 |

> 保密相关违纪同时适用《保密协议管理制度》。

### 3.5 职场行为纪律

| 行为 | 处分 |
|------|------|
| 工作场所争吵、不文明行为 | 口头/书面警告 |
| 职场骚扰、霸凌 | 记过及以上，情节严重者解除 |
| 打架斗殴 | 解除劳动合同 |
| 伪造学历、履历等入职欺诈 | 解除劳动合同（合同自始无效） |

---

## 4. 处分程序

1. **调查取证**：发现违纪线索后，由 HR/法务部启动调查，收集证据（记录、监控、证人证言等）。
2. **告知与申辩**：将违纪事实书面告知员工，员工有权在 **5个工作日内** 提出书面申辩。
3. **处分决定**：综合事实与申辩意见，由 HR 会同法务部提出处分意见，按权限审批。
4. **送达**：处分决定书面送达员工本人，员工签收（拒签的可通过见证、邮寄等方式送达）。
5. **归档**：处分记录存入员工档案。

> 涉及解除劳动合同的，须经法务部审核合规性，并按《劳动合同管理办法》执行，必要时通知工会。

---

## 5. 申诉机制

- 员工对处分有异议，可在收到处分决定后 **7个工作日内** 向 HR 提交《处分申诉书》。
- 由 HR、法务部及员工所在部门外的管理者组成复核小组，在 **10个工作日内** 复核并答复。
- 申诉不影响处分决定的执行，但复核认定处分不当的，予以撤销或变更。

---

## 6. 处分的撤销与影响消除

- 口头警告、书面警告：自处分之日起 **6个月** 内无新违纪的，影响自动消除。
- 记过：自处分之日起 **12个月** 内表现良好的，可申请消除档案记录。
- 降级/解除：不予撤销（除经申诉认定错误）。

---

## 7. 常见问题（FAQ）

**Q1：被口头警告会影响我的工资吗？**
A：口头警告本身不直接扣款，但会记录在案并影响当月绩效评分。伴随的考勤扣款按《考勤政策》执行。

**Q2：公司说我严重违纪要辞退，我能申辩吗？**
A：可以。处分决定前公司须书面告知违纪事实，你有5个工作日提出书面申辩。处分后还可在7个工作日内申诉。

**Q3：旷工几天会被开除？**
A：连续旷工3天，或一年内累计旷工7天，可解除劳动合同。

**Q4：处分记录会跟着我一辈子吗？**
A：不会。警告类6个月、记过12个月后表现良好可消除影响。降级和解除不予撤销。

**Q5：我觉得处分太重了怎么办？**
A：在收到决定后7个工作日内提交申诉，复核小组会重新评估处分是否适当。

## 8. 附则

- 本条例由法务部负责解释，与人力资源部会签执行。
- 本条例与《劳动合同管理办法》《保密协议管理制度》《考勤政策》配套使用。
- 处分涉及解除劳动合同的，以国家法律法规为准。
- 本条例自2026年6月1日起生效。
