# 采购流程规范

**版本：** v3.0 | **生效日期：** 2024年1月1日 | **归口部门：** 财务部
**最新修订：** 2026年5月30日 | **会签部门：** 行政部、信息技术部

---

## 1. 总则

为规范公司采购行为、控制采购成本、防范廉政风险，确保采购活动公开、公平、公正，特制定本规范。本规范适用于公司所有以公司资金进行的物资、设备、服务采购。

---

## 2. 采购分类

| 类型 | 说明 | 典型示例 |
|------|------|---------|
| 日常采购 | 办公用品、低值易耗品 | 文具、纸张、打印耗材 |
| 设备采购 | 固定资产类 | 电脑、显示器、服务器、办公家具 |
| 服务采购 | 无形服务类 | 软件许可、咨询、培训、外包服务 |
| 工程采购 | 装修、基建 | 办公室装修、机房建设 |
| 应急采购 | 紧急需求 | 走绿色通道，事后补流程 |

---

## 3. 审批权限

| 单次金额 | 审批人 |
|---------|--------|
| ≤ 2000元 | 直属上级 |
| 2000 - 10000元 | 直属上级 + 部门总监 |
| 10000 - 50000元 | 部门总监 + 财务总监 |
| 50000 - 200000元 | 财务总监 + 分管副总裁 |
| ＞ 200000元 | 管理层集体审批 |

---

## 4. 采购方式与比价要求

| 采购方式 | 适用金额 | 要求 |
|---------|---------|------|
| 直接采购 | ＜ 10000元 | 由申请部门自行选定，注明理由 |
| 比价采购 | 10000 - 50000元 | 至少3家供应商书面报价，择优选定 |
| 询价采购 | 50000 - 200000元 | 至少3家正式询价，财务部参与 |
| 公开招标 | ≥ 200000元 | 公开招标，成立评标小组 |

### 4.1 独家供应商

确属独家供应或唯一来源的，需提供书面《独家供应说明》，经财务总监批准后可免比价。

### 4.2 框架协议

对长期、重复性采购（如办公用品），可与供应商签订年度框架协议，约定价格与服务标准，期内无需逐次比价。

---

## 5. 采购执行流程

1. **需求申请**：申请部门在 OA 提交《采购申请单》，注明品名、规格、数量、预算、期望到货时间。
2. **审批**：按金额逐级审批（见第3节）。
3. **询价/比价**：按采购方式执行，留存报价记录。
4. **下单签约**：与选定供应商签订采购合同或下采购订单（合同走《合同审批流程》）。
5. **到货验收**：申请部门验收。
6. **入库登记**：固定资产贴标签并录入资产管理系统。
7. **付款结算**：财务按合同账期付款。

---

## 6. 验收与付款

### 6.1 验收要求

- 到货后由申请部门负责人或指定人员验收，核对品名、规格、数量与采购订单一致。
- 固定资产须贴资产标签并录入 IT 资产管理系统（由信息技术部执行）。
- 验收合格后签署《验收确认单》，作为付款依据。
- 发现质量问题须在 **3个工作日内** 向供应商提出，逾期视为验收通过。

### 6.2 付款规则

- 验收合格后 **30天内** 付款（合同约定更优的按合同执行）。
- 付款须凭：验收确认单 + 供应商发票 + 采购合同/订单。
- 预付款须经财务总监审批，且不超过合同总额的 **30%**。

---

## 7. 采购纪律

- **禁止拆分采购**：不得将单次采购拆分为多笔以规避审批权限。
- **禁止指定关联供应商**：采购人员及其直系亲属不得参与相关供应商的评选。
- **禁止收受回扣**：采购人员不得以任何形式收受供应商礼品、回扣、佣金（价值超过200元须申报）。
- **利益冲突申报**：采购人员如与候选供应商存在关联关系，须主动申报并回避。
- 违规者按《员工纪律处分条例》处理，情节严重的移送司法机关。

## 8. 常见问题（FAQ）

**Q1：我要买一台2500元的显示器，需要谁审批？**
A：需要直属上级 + 部门总监双签（2000-10000元区间）。

**Q2：已有框架协议的供应商还需要比价吗？**
A：不需要。框架协议期内按约定价格直接采购，无需重新比价。

**Q3：紧急采购怎么走？**
A：先电话/微信请示直属上级和财务部口头同意，完成采购后 **24小时内** 补交 OA 申请，注明紧急原因。

**Q4：采购的东西验收后发现有问题，怎么处理？**
A：在3个工作日内书面通知供应商，要求退换货或赔偿。超期视为验收通过，后续维权难度增加。
