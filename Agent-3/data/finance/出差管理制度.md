# 出差管理制度

**版本：** v3.0 | **生效日期：** 2024年1月1日 | **归口部门：** 财务部
**最新修订：** 2026年5月30日 | **会签部门：** 人力资源部、行政部

---

## 1. 总则

### 1.1 目的

为规范公司因公出差管理，明确出差申请、交通住宿标准及差旅费报销规则，控制差旅成本、保障出差人员权益，特制定本制度。

### 1.2 适用范围

本制度适用于公司全体正式员工因公出差（含国内、国（境）外）。出差指员工离开常驻工作城市，前往外地执行公务并当日无法返回，或单程距离超过 **50公里** 的公务活动。

### 1.3 出差定义与分类

| 类型 | 定义 | 适用规则 |
|------|------|---------|
| 短途出差 | 当日往返，单程＞50公里 | 不享受住宿补助，餐饮补助按50%发放 |
| 普通出差 | 1~7个自然日 | 按标准享受交通、住宿、餐饮补助 |
| 长期出差 | 连续＞7个自然日 | 7天以上部分住宿标准下调10%，需部门总监审批 |
| 境外出差 | 出国（境）公务 | 按境外标准执行，详见第7节 |

## 2. 出差申请

### 2.1 申请流程

1. 员工提前至少 **2个工作日** 在 OA 系统提交《出差申请单》，注明目的地、事由、预计起止日期、预算金额、交通方式。
2. 直属上级审批（1个工作日内）。
3. 出差天数 ≥ 5天 或 预算 ≥ 10000元：部门总监加签。
4. 境外出差：须分管副总裁审批，并提前 **5个工作日** 申请。

### 2.2 紧急出差

因突发业务需紧急出差的，可先电话/微信请示直属上级口头同意后出发，并在 **出发后24小时内** 补交 OA 申请，注明紧急原因。未补交申请的，差旅费不予报销。

### 2.3 出差变更与取消

- 行程变更（延期、改地点）需在 OA 提交《出差变更申请》。
- 已订票务因公务取消产生的退改签费用，凭凭证可据实报销；因个人原因产生的退改签费用自理。

## 3. 交通标准

| 职级 | 火车 | 飞机 | 市内交通 |
|------|------|------|---------|
| P1-P4 | 高铁/动车二等座 | 经济舱 | 实报实销 ≤ 100元/天 |
| P5-P6 | 高铁/动车一等座 | 经济舱 | 实报实销 ≤ 200元/天 |
| P7+ | 高铁/动车商务座 | 商务舱（飞行＞4小时可选） | 实报实销 |

### 3.1 交通方式选择原则

- **就近就经济原则**：优先选择高铁，行程＞800公里或高铁耗时＞6小时可选择飞机。
- **提前预订**：机票应提前预订以享受折扣，非紧急情况下临时全价票需说明原因。
- **统一订票**：机票、火车票原则上通过公司差旅平台统一预订，自行购票需提供完整票据。
- **自驾出差**：经批准可自驾，按实际里程 **1.5元/公里** 报销油费过路费（需提供行车记录与过路费票据），不再另行报销其他交通费。

## 4. 住宿标准

| 城市等级 | 标准 | 城市范围 |
|---------|------|---------|
| 一线 | ≤ 600元/晚 | 北京、上海、广州、深圳 |
| 新一线 | ≤ 450元/晚 | 杭州、成都、武汉、南京、苏州等 |
| 二线 | ≤ 400元/晚 | 各省会及计划单列市 |
| 其他 | ≤ 300元/晚 | 其余城市 |

### 4.1 住宿规则

- 住宿费凭酒店发票据实报销，不超过对应城市标准。
- 超标准住宿需提前申请并说明原因（如展会期间酒店紧张），经部门总监批准后可上浮不超过20%。
- 同性别同事同地出差，鼓励合住标准间以节约成本，合住的可各自享受标准的80%作为补贴激励。
- 出差地有公司分支机构或合作宿舍的，应优先入住，不再报销酒店费用。

---

## 5. 餐饮补助

- 出差期间每日餐饮补助 **100元**（无需发票，按出差自然天数计发）。
- 当天往返不享受住宿补助，餐饮补助按 **50%**（50元/天）发放。
- 出差期间如有公司或客户统一安排用餐的，相应餐次的补助按比例扣减。
- 客户招待费用单独走业务招待费报销，不计入个人餐饮补助。

---

## 6. 报销要求

### 6.1 报销时限与材料

- 出差结束后 **5个工作日内** 在 OA 提交差旅费报销，超期需直属上级特批。
- 报销材料：出差申请单（已审批）、交通票据、住宿发票、行程单、出差总结（≥3天需附）。

### 6.2 差旅费构成

差旅费报销单应分项列明：交通费、住宿费、餐饮补助、市内交通、其他（如会议注册费、签证费等）。

### 6.3 预借差旅费

- 长途或境外出差可申请预借差旅费，金额不超过预算的 **80%**。
- 出差结束后须在5个工作日内凭实际票据核销，多退少补。

## 7. 境外出差

### 7.1 审批与准备

- 境外出差须提前 **5个工作日** 申请，由分管副总裁审批。
- 涉及签证、保险的，由行政部协助办理；出差人须自行确保证件有效期。
- 公司为境外出差人员统一购买境外旅行意外险。

### 7.2 境外标准

| 区域 | 住宿标准（每晚） | 餐饮补助（每日） |
|------|----------------|----------------|
| 港澳台 | ≤ 800港币/新台币等值 | 200元人民币 |
| 东南亚 | ≤ 600元人民币等值 | 250元人民币 |
| 欧美/日韩/澳新 | ≤ 1200元人民币等值 | 400元人民币 |
| 其他地区 | 参照欧美标准 | 300元人民币 |

- 境外费用按费用发生当日中国银行外汇牌价折算人民币报销。
- 境外交通、通讯等其他费用据实报销，需提供票据。

## 8. 常见问题（FAQ）

**Q1：出差当天往返，能报住宿吗？**
A：不能。当天往返不产生住宿，仅按50%发放餐饮补助（50元）。

**Q2：我职级是P5，能坐飞机商务舱吗？**
A：不能。P5-P6 飞机为经济舱，仅 P7 及以上飞行超过4小时可选商务舱。

**Q3：出差期间周末的餐补怎么算？**
A：餐饮补助按出差自然天数计发，包含出差期间的周末，无需区分工作日。

**Q4：酒店超标了一点点能报吗？**
A：超标部分原则上自理。如确因当地酒店紧张等客观原因，需提前申请并经总监批准，最多上浮20%。

**Q5：可以自己订机票再报销吗？**
A：原则上通过公司差旅平台统一预订。自行购票需提供完整票据（行程单+发票），且不得高于同期统一预订价格。

**Q6：出差顺路办私事产生的费用怎么处理？**
A：私人行程产生的交通、住宿、餐饮费用一律自理，差旅费仅覆盖公务部分。

- 本制度由财务部负责解释，与人力资源部、行政部会签执行。

- 本制度自2024年1月1日起生效，废止旧版《出差管理制度 v2.0》。
