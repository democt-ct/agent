# 报销管理制度

**版本：** v3.0 | **生效日期：** 2024年1月1日 | **归口部门：** 财务部
**最新修订：** 2026年5月30日 | **审核人：** 财务总监 | **批准人：** CFO

---

## 1. 总则与适用范围

### 1.1 制定目的

为规范公司费用报销行为，明确报销标准、审批权限与发票合规要求，保障公司资金安全、提升财务处理效率，特制定本制度。

### 1.2 适用范围

本制度适用于公司**全体正式员工**（含试用期员工）因公产生的各类费用报销。下列情形**不适用**本制度：

- 劳务派遣人员、外包人员的费用（由其所属单位承担或按合作合同约定执行）。
- 实习生因公费用（参照执行，但需由带教导师与部门负责人双重确认）。
- 涉及对外投资、并购、重大资产购置的支出（按《重大事项审批制度》单独执行）。

### 1.3 基本原则

- **真实性原则**：所有报销必须基于真实发生的因公费用，严禁虚报、冒领、重复报销。
- **相关性原则**：费用须与公司经营业务直接相关，与个人消费混同的费用不予报销。
- **及时性原则**：费用应在发生后规定时限内提交报销，逾期需特批。
- **合规性原则**：发票、单据须符合国家税法及公司财务规范。

## 2. 报销类型与限额

| 类型 | 限额 | 说明 |
|------|------|------|
| 差旅费 | 实报实销 | 交通+住宿+餐饮补助，按《出差管理制度》标准执行 |
| 办公用品 | 单次≤500元 | 超出需提前走采购申请流程 |
| 业务招待费 | 人均≤200元 | 需注明招待对象、单位、人数及事由 |
| 培训费 | 单次≤3000元 | 需部门负责人+HR双重审批 |
| 交通费（市内） | 月累计≤800元 | 因公打车、公共交通，需注明起讫地点与事由 |
| 通讯补贴 | 按职级标准 | 详见第6节职级标准表 |
| 会议费 | 单次≤5000元 | 含场地、餐饮、物料，超出需总监审批 |
| 其他 | 单次≤1000元 | 需详细说明费用性质与必要性 |

> **说明**：限额为单次报销上限，超出限额的费用须在费用发生**前**提交专项申请，经审批后方可报销。事后补报超限费用原则上不予受理。

## 3. 报销流程

### 3.1 标准流程

1. **填单**：员工在 OA 系统提交《费用报销单》，逐笔录入费用明细，上传发票照片/扫描件。
2. **直属上级审批**（1个工作日内）：核对费用真实性与业务相关性。
3. **分级加签**：
   - 单笔或单次累计 ≥ 5000元：财务部加签复核。
   - 单笔或单次累计 ≥ 20000元：财务总监审批。
   - 单笔或单次累计 ≥ 50000元：CFO 审批。
4. **财务审核**：财务专员审核发票合规性、金额准确性、预算余额。
5. **出纳打款**：审批通过后 **3个工作日内** 打款至员工工资卡。

### 3.2 流程时效

| 环节 | 时效要求 | 超时处理 |
|------|---------|---------|
| 直属上级审批 | 1个工作日 | 自动提醒，超3日升级至部门总监 |
| 财务审核 | 2个工作日 | 退回补正的需注明原因 |
| 出纳打款 | 审批通过后3个工作日 | 遇银行结算高峰可顺延，但不超过5个工作日 |

> 每月 **25日至月末** 为财务结账期，此期间提交的报销单顺延至次月初处理。

## 4. 发票与单据要求

### 4.1 发票基本要求

- **发票类型**：增值税普通发票或增值税专用发票（一般纳税人取得专票可抵扣进项税，优先取得专票）。
- **发票抬头**：必须为 **XX科技有限公司**（全称），不接受个人抬头或简称。
- **纳税人识别号**：开票时须提供公司税号（财务部统一下发，见 OA「报销帮助」栏）。
- **提交时限**：发票日期需在费用发生后 **30天内** 提交报销，跨月不跨年。
- **电子发票**：需打印纸质件并由报销人签字；同一张电子发票不得重复报销，系统会校验发票号查重。

### 4.2 不予受理的发票情形

- 发票抬头错误、税号缺失或错误。
- 发票项目与实际消费不符（如餐饮开成办公用品）。
- 发票模糊、涂改、缺联、连号异常。
- 定额发票、收据等无法验真的凭证（特殊情况需附情况说明并经财务总监特批）。
- 超过开票日期180天仍未提交的发票。

### 4.3 无发票费用处理

部分确无法取得发票的费用（如个人交通补助、误餐费、小额过路过桥费），可凭《无票费用说明单》报销，但：

- 单笔不超过 **200元**，月累计不超过 **1000元**。
- 需注明费用发生时间、地点、事由，并由直属上级签字确认。
- 财务部有权抽查核实，发现虚报的按违规处理。

## 5. 特殊场景报销细则

### 5.1 业务招待费

- 报销时须填写《招待费明细表》，注明：被招待方单位、姓名、人数、招待事由、本方陪同人员。
- 人均标准：普通客户 ≤ 200元/人，重要客户（需总监确认）≤ 400元/人。
- 严禁将个人聚餐、团队团建以招待费名义报销（团建走专项预算）。
- 单次招待 ≥ 3000元的，需提前在 OA 提交《招待申请单》。

### 5.2 跨部门/项目费用分摊

- 涉及多部门共同承担的费用（如联合项目、共享会议），报销时需在备注中注明分摊比例。
- 财务按比例分别计入各部门成本，分摊方案需相关部门负责人会签确认。

### 5.3 代付与垫付

- 员工因公垫付的大额费用（≥ 10000元），可在费用发生前申请 **预借款**，事后凭发票冲抵。
- 预借款须在事项结束后 **15个工作日内** 完成核销，逾期未核销的从工资中扣回。

### 5.4 外币费用报销

- 因公出境产生的外币费用，按**费用发生当日**中国银行外汇牌价折算为人民币报销。
- 需附银行结算单或信用卡账单作为汇率佐证。

## 7. 违规处理

| 违规行为 | 处理措施 |
|---------|---------|
| 虚假报销（虚开发票、虚构费用） | 全额退回款项 + 书面警告，情节严重者解除劳动合同并追究法律责任 |
| 重复报销 | 退回重复部分，首次警告，再次记入个人档案 |
| 超期提交 | 原则上不予报销，特殊情况需部门总监特批 |
| 发票不合规 | 退回补正，同一报销单累计退回超3次记入个人档案 |
| 私人消费冒充公费 | 全额退回 + 通报批评 |
| 审批人失职审批 | 视情节予以提醒、通报，造成损失的承担连带责任 |

## 8. 常见问题（FAQ）

**Q1：发票丢失了还能报销吗？**
A：增值税专票丢失需取得开票方的《丢失证明》及记账联复印件（加盖公章）；普票丢失可凭复印件加开票方盖章报销，但财务有权拒绝无法验真的凭证。建议养成及时拍照存档的习惯。

**Q2：上个月的费用这个月才报，可以吗？**
A：可以，只要发票在开具后30天内、且不跨年度。跨月报销建议在备注说明原因。

**Q3：报销款多久到账？**
A：审批全部通过后3个工作日内打款至工资卡。若遇月末结账期（25日后），顺延至次月初。

**Q4：能用别人的发票帮我报销吗？**
A：不可以。报销发票必须为本人因公实际消费产生，冒用他人发票属虚假报销，按违规处理。

**Q5：餐饮发票一定要明细吗？**
A：业务招待餐饮发票需附消费清单（菜品明细）；个人误餐费定额标准内可不附清单，但需说明事由。

**Q6：预借款没用完怎么办？**
A：核销时多余款项需退回公司账户或从下月工资中扣回，预借款不得挪作他用。
