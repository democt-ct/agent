# 请假审批流程

## 1. 概述

本制度适用于公司全体员工（含试用期员工、正式员工，**不含实习生及劳务派遣人员**），旨在规范各类请假的申请、审批、计算及工资处理，确保考勤管理的公平性与合规性。所有请假行为均须严格遵循本流程，**未经批准擅自缺勤将按旷工处理**。人力资源部为本制度的归口管理部门，负责日常解释、培训及违规处理。

## 2. 请假类型总览

公司提供以下假期类型，具体适用条件及所需材料详见各分项说明。

| 请假类型 | 简称 | 申请渠道 | 审批层级（≤3天） | 审批层级（>3天） | 最长天数 | 必备材料 |
|---------|------|---------|-----------------|-----------------|---------|---------|
| 年假 | 年 | OA系统 → 假期管理 | 直属上级 | 直属上级→部门总监 | 按工龄计算（5~15天） | 无 |
| 病假 | 病 | OA系统 → 假期管理 | 直属上级 | 直属上级→HRBP | 连续≤30天 | 二级及以上医院证明、病历 |
| 事假 | 事 | OA系统 → 假期管理 | 直属上级 | 直属上级→部门总监→HRBP | 单次≤15天，年度累计≤30天 | 无 |
| 婚假 | 婚 | OA系统 → 假期管理 | 直属上级 | 直属上级→HRBP | 13天（含周末及节假日） | 结婚证（建议上传） |
| 产假 | 产 | OA系统 → 假期管理 | HRBP | HRBP→HRD | 98天+60天奖励假（视地区政策） | 生育服务登记证、医院证明 |
| 陪产假 | 陪 | OA系统 → 假期管理 | 直属上级 | 直属上级→部门总监 | 15天（含周末及节假日） | 出生医学证明 |
| 丧假 | 丧 | OA系统 → 假期管理 | 直属上级 | 直属上级→HRBP | 3天（父母/配偶/子女）；1天（祖父母/兄弟姐妹/配偶父母） | 死亡证明或讣告（建议上传） |

> **备注**：所有假期均须在OA系统中选择对应类型提交，不得以“其他”替代。审批层级中的“>3天”指累计请假天数超过3天（含本数）。如遇法定节假日与假期重合，按第4.2条规则处理。

### 2.1 年假详细说明

- **工龄计算**：以社保缴纳记录为准。入职满1年不足10年：5天/年；满10年不足20年：10天/年；满20年：15天/年。**工龄从首次社保缴纳之日起累计计算，非本公司工龄视同有效**。
- **最小取假单元**：0.5天（上午或下午）。当年未休完按日工资300%折算（仅限离职或政策强制要求）。**当年年假未休满1天的部分（如0.5天）不折算**。
- **不可预支**：当年年假额度仅可在入职满6个月后使用，试用期员工不享受年假。**例如**：员工李某于2026年3月1日入职，2026年9月1日起可申请年假，额度按社保工龄计算。
- **跨年结转**：最多结转5天至次年第一季度末（即次年3月31日），逾期未休则自动失效。**例如**：2026年剩余7天年假，最多结转5天至2027年使用，剩余2天自动清零。

### 2.2 病假详细说明

- **材料要求**：必须上传二级及以上公立医院的挂号单、诊断证明及病历首页。连续请假≥5天的，需同时提供病假条（写明建议休息天数）。**急诊病假可先就医后补材料**，但必须在请假后24小时内电话报备直属上级。
- **证明时效**：病假结束后3个工作日内补传证明，逾期视为事假或旷工（需HR确认）。**若因病无法操作OA，可委托同事代为提交**，并附书面授权说明。
- **特殊情形**：因病需住院手术的，可凭住院通知书申请最长30天病假，后续需提供出院小结。**住院期间遇周末或法定假日，不另扣减额度**。

### 2.3 事假详细说明

- **事假期间无工资**，且必须写明具体事由（如“办理个人房产手续”“陪同家属就医”等），禁止以“个人事务”“家里有事”等模糊理由申请。**未写明具体事由或理由过于模糊的，审批人有权驳回**。
- **连续事假超过7天**，须经部门总监及HRBP双签同意。**双签指两人均需在OA上点击“同意”，任一方驳回则申请不通过**。
- **年度累计事假超过30天**，视为员工主动放弃年终绩效奖金资格。**年度累计指自然年（1月1日~12月31日）内所有事假天数的总和**。

## 3. 申请流程

### 3.1 标准申请步骤

1. **员工登录OA系统**：进入“假期管理”模块，点击“新建请假申请”。**建议使用Chrome或Edge浏览器，避免兼容性问题**。
2. **填写信息**：选择请假类型 → 填写起止日期（精确到半天） → 输入请假事由（50字以上详细描述） → 选择审批人（默认为直属上级，可修改备注）。**事由栏需包含：请假原因、工作交接安排、紧急联系人及电话**。
3. **上传附件**：根据上表“必备材料”要求上传证明文件。文件格式限定PDF/JPG，单文件≤10MB。**如材料为多页，建议合并为一个PDF文件**。
4. **提交与审批**：提交后，系统自动通过OA消息、邮件及企业微信三通道通知审批人。审批人须在**24小时（自然小时）内**完成审批（含驳回、退回或通过）。**超时处理见第3.3条**。
5. **结果反馈**：审批通过后，系统自动更新考勤日历，申请人可在“我的假期”查看已生效记录。**若审批被驳回，需重新提交并说明修改内容**。
6. **假期执行**：员工按时休假，休假期间需保持工作邮箱/电话畅通（紧急事务可转接）。**建议设置自动回复邮件，说明休假起止时间及紧急联系人**。

### 3.2 紧急请假处理

- **突发疾病或意外事故**：员工可先行电话/微信通知直属上级，说明情况并预估请假时长。随后在**24小时内**登录OA系统提交“紧急补办”申请，注明事由及“紧急补办”标签。**若无法登录OA，由直属上级代提交，员工事后补签**。
- **材料补充期限**：证明材料须在休假结束后的**3个工作日内**补齐，否则按事假或旷工处理。**示例**：员工张某突发高烧，早上8:00电话告知上级，当日无法上班。张某当天未提交OA申请，次日提交并上传急诊病历，审批人确认后按病假处理。
- **因公受伤**：需同时提交工伤认定申请，按【工伤管理制度】执行，不适用本紧急请假流程。

### 3.3 审批时效与超时处理

- **标准时效**：审批人应在收到申请后**24小时（自然小时）内**完成审批。超30分钟未审者，系统自动发送首次催办提醒（OA消息+企业微信）；超48小时仍未审批的，视为**系统自动通过**（须HRBP确认该假期类型及天数符合制度，否则可人工驳回）。**自动通过后，HRBP有权在3个工作日内进行复核**。
- **审批人休假/无法操作**：员工应在提交前将审批人设置为**代理人**（在OA系统“审批设置”中指定）。若未设置且审批人长期不操作，可直接联系HRBP介入处理。**示例**：员工王某的直属上级出差未设置代理，王某请事假3天，系统超48小时未审批自动通过，HRBP复核后认为理由合理，维持通过。

## 4. 请假时长计算规则

### 4.1 基本规则

- **按自然日计算**：从请假开始日的0:00到结束日的23:59，含所有周末及法定节假日。**但年假、病假、事假的额度扣除以工作日为准**（见下文）。
- **半天假**：上午时段（8:30-12:00）或下午时段（13:00-18:00），每次最多计4小时。禁止跨时段申请（如上午8:30-下午13:00不属于半天）。**示例**：员工下午请假半天，则13:00前需正常打卡，13:00后无需到岗。
- **最小请假单位**：0.5天。单次请假连续时长可精确到半天，不可申请小时假（特殊情况如迟到早退使用考勤补卡流程）。**迟到30分钟内可通过OA“迟到申请”补卡，次数计入月度考核**。

### 4.2 跨周期处理

- **跨周末/节假日**：连续请假中包含的周末不额外扣减年假额度（如周五+下周一请年假，中间的周六日不计入年假天数，但休假状态持续）。**病假、事假同样规则：实际占用工作日的部分才扣除额度**。
- **跨月休假**：请假起止日跨越两个月份的，按实际休假天数分别计入各月考勤。工资发放按出勤天数比例计算。**例如**：员工请病假从1月25日至2月5日，1月25-31日（5工作日）计入1月，2月1-5日（3工作日）计入2月。
- **与法定假重合**：员工在婚假、产假、丧假期间遇法定节假日的，该节假日不计入假期天数（即婚假13天内含法定假则顺延）。**年假、病假、事假遇法定节假日的，该节假日仍计入请假天数**。

## 5. 请假撤销与变更

- **撤销**：审批前，员工可在OA自行撤回申请。审批通过后，需撤销假期的，必须由员工发起“假期撤销”的独立审批（流程同请假，审批人不变）。撤销成功后方可申请新假期。**撤销申请需在休假开始前提交，休假开始后不可撤销**。
- **变更**：请假期间无法变更假期类型（如年假改病假），需先撤销原假期，再重新申请。若因突发疾病需将年假转为病假，可在销假后3天内提交证明材料，经HRBP确认后调整。**示例**：员工原计划休年假3天，第2天突发疾病住院，可先电话报备上级，假期结束后提交病假证明，HRBP将年假调整为病假，已休的1天年假退还。

## 6. 假期工资说明

| 假期类型 | 工资计算方法 | 举例说明 |
|---------|-------------|---------|
| 年假 | 全额发放基本工资+岗位津贴 | 月薪12000元（基本工资8000+岗位津贴4000），休3天年假，发放12000/21.75×3 = 1655元（按统一基数计算） |
| 病假 | 按工龄比例发放基本工资（不含绩效、津贴）：入职不满2年发70%，满2年不满5年发80%，满5年及以上发90% | 月基本工资8000元，工龄3年，休病假2天，发放8000/21.75×2×80% = 588.5元 |
| 事假 | 扣除当日全额工资（基本工资+岗位津贴+绩效工资），最低不低于当地最低工资标准 | 月薪15000元（基本工资8000+岗位津贴4000+绩效3000），请事假1天，扣除15000/21.75 = 689.7元 |
| 婚假 | 全额发放（含基本工资、岗位津贴、绩效） | 与年假同 |
| 产假 | 按生育津贴规定由社保局拨付，不足部分公司补齐至全额工资 | 津贴低于月薪时，公司补差 |
| 丧假 | 全额发放，同婚假 |  |

> **注意**：病假工资计算以**基本工资**为基数，不含加班费、餐补、交通补贴等。请假当天若因公司安排调休则不扣工资。**日工资基数 = 月工资总额 ÷ 21.75**。

## 7. 违规处理

- **未申请或未批准缺勤**：视为旷工，扣当日双倍工资，并记入年度绩效考核。连续旷工≥3天或年度累计≥5天，公司有权解除劳动合同。**旷工期间不缴纳社保个人部分，公司部分仍承担**。
- **提供虚假材料**：如伪造医院证明、结婚证等，一经查实取消当期请假资格，给予**书面警告**处分，全公司通报，且半年内禁止申请任何带薪假期。**情节严重者，公司保留追究法律责任的权利**。
- **超期未归且未续假**：请假到期后未返岗，且未在24小时内向直属上级和HRBP说明情况者，按自动离职处理，工资发至实际工作日止。**自动离职不影响公司追索因其缺勤造成的经济损失**。
- **请假理由与实际不符**：经查实（如事假外出旅游），按旷工处理，并追回已发工资。**多次违规者，列入公司黑名单**。

## 8. 常见问题解答（FAQ）

**Q1：我是新入职员工，试用期内可以请年假吗？**  
A：试用期员工不享受年假，若确有需要，可按事假申请。转正后统一按社保工龄计算年假天数。

**Q2：我的直属上级出差/休假，无法审批，怎么办？**  
A：您应在提交前设置审批代理（在OA系统审批设置中指定本部门副职或HRBP）。若未设置，可直接联系HRBP介入处理。

**Q3：病假证明可以后补吗？最晚什么时候？**  
A：可以后补，但最晚须在假期结束后3个工作日内上传至OA系统对应申请单。逾期未补者，原病假将自动转为事假（需HRBP确认），并按事假扣薪。

**Q4：我请半天假，是需要上满4小时，还是上上午或下午各半个？**  
A：半天假对应固定时段：上午（8:30-12:00）或下午（13:00-18:00）。例如，您申请“下午半天假”，则下午不需到岗，但上午须正常打卡。不可以在上午上3小时、下午上1小时凑成半天。

**Q5：跨周末的年假，周末时间算在年假内吗？**  
A：不算。连续请假中的周末不计入年假额度，但您仍可正常休息。例如周五请假年假1天，下周一请假年假1天，中间的周六日不需要额外申请假期。

**Q6：请事假期间，我的社保和公积金如何缴纳？**  
A：事假期间社保和公积金仍由公司正常缴纳，个人应缴部分从后续工资中扣除。连续事假超过1个月的，需提前与HR确认缴费安排。

**Q7：如何查询我的剩余年假天数？**  
A：登录OA系统 → 进入“我的假期” → 查看年假余额。原则上每年1月1日自动更新额度，当年未休完的按结转规则执行。

**Q8：我可以用加班调休抵扣事假吗？**  
A：加班调休仅可抵扣等时长的补休，不可直接转换为事假。如需调休，请使用OA“加班调休”流程单独申请。**调休有效期自加班之日起6个月内有效**。

**Q9：病假工资计算时，工龄如何界定？**  
A：工龄以在本公司连续社保缴纳记录为准，入职前在其他单位的社保不累计。例如员工王某2024年1月入职，2026年5月病假，此时工龄为2年4个月，属于“满2年不满5年”，按80%发放。

**Q10：产假期间公司补差的工资何时发放？**  
A：公司收到生育津贴后，在次月工资发放日统一核算补差。津贴拨付前，产假期间工资暂按基本工资发放，待津贴到账后多退少补。
