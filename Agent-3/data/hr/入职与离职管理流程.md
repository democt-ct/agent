# 入职与离职管理流程

**版本：** v1.0 | **生效日期：** 2026年6月1日 | **归口部门：** 人力资源部

---

## 1. 总则

本流程规范公司员工从入职、转正到离职的全生命周期管理，确保人员流动有序、交接清晰、风险可控。适用于公司全体员工。

---

## 2. 入职管理

### 2.1 入职前准备

- HR 在员工入职前发送《入职通知书》，告知报到时间、地点、所需材料。
- 所需材料：身份证、学历学位证、离职证明（如有）、银行卡、体检报告、一寸照片、个人信息表。
- IT 部门提前准备办公设备与账号（见《设备申领管理办法》）。

### 2.2 入职当天流程

1. 报到签到，提交入职材料。
2. 签订劳动合同与保密协议（见《劳动合同管理办法》《保密协议管理制度》）。
3. 领取办公设备、工牌、门禁卡。
4. 开通企业邮箱、OA、各业务系统账号。
5. 参加入职引导（公司介绍、制度宣讲、安全培训）。
6. 与直属上级、团队成员见面，了解岗位职责。

### 2.3 试用期管理

| 合同期限 | 试用期 |
|---------|-------|
| 3个月以上不满1年 | 不超过1个月 |
| 1年以上不满3年 | 不超过2个月 |
| 3年以上或无固定期限 | 不超过6个月 |

- 试用期工资不低于转正后的80%，且不低于当地最低工资。
- 试用期满前由直属上级发起转正评估，HR 审核。
- 试用期不符合录用条件的，公司可依法解除劳动合同（需说明理由并留存证据）。

### 2.4 转正流程

1. 试用期满前 **5个工作日**，员工提交《转正申请及自我评估》。
2. 直属上级评估并填写《转正评估表》。
3. 部门负责人审批。
4. HR 复核，办理转正手续，调整薪资至转正标准。

## 3. 离职管理

### 3.1 离职类型

| 类型 | 说明 | 提前通知期 |
|------|------|-----------|
| 主动辞职 | 员工提出 | 试用期提前3天，正式员工提前30天书面通知 |
| 协商解除 | 双方协商一致 | 按协议约定 |
| 合同到期 | 劳动合同期满不续签 | 到期前30天告知 |
| 公司解除 | 因违纪、不胜任等 | 按《劳动合同法》执行 |

### 3.2 离职申请流程

1. 员工在 OA 提交《离职申请单》，注明离职原因与期望离职日期。
2. 直属上级审批，安排工作交接。
3. 部门负责人审批。
4. HR 进行离职面谈，了解离职原因。
5. 完成工作交接与物品归还。
6. HR 办理离职手续，结算薪资，开具离职证明。

### 3.3 工作交接

离职员工须在离职前完成以下交接，由接收人和上级签字确认：

- **工作交接**：在办项目、客户资源、文档资料、密码权限、未尽事项清单。
- **物品归还**：办公设备（见《设备申领管理办法》归还流程）、工牌、门禁卡、钥匙、公司文件。
- **账号回收**：IT 部门停用/移交各系统账号、邮箱（离职当日生效）。
- **财务结清**：预借款核销、报销结清、无欠款证明。

### 3.4 离职结算

- 最后工作日后 **15个工作日内** 完成薪资结算。
- 结算内容：当月工资、未休年假补偿（见《年假管理制度》）、绩效奖金、扣除项（如未核销预借款、超休年假）。
- HR 在离职手续办结后开具《离职证明》。

---

## 4. 离职证明

离职证明载明：员工姓名、身份证号、在职岗位、入职与离职日期、解除/终止劳动合同的事实。**不包含**对员工不利的主观评价。员工可凭离职证明办理失业登记、新单位入职等。

## 5. 常见问题（FAQ）

**Q1：我想辞职，需要提前多久通知？**
A：试用期提前3天，转正后提前30天以书面形式（OA 离职申请或书面信函）通知，以便安排交接。

**Q2：没做完30天就能走吗？**
A：法律规定提前30天通知，提前离开需与公司协商一致。未办交接擅自离职可能影响离职证明开具及结算。

**Q3：离职时年假没休完有补偿吗？**
A：有。未休年假按日薪200%折算补偿，详见《年假管理制度》离职结算条款。

**Q4：离职工资什么时候发？**
A：最后工作日后15个工作日内完成结算并发放。

**Q5：试用期被辞退有补偿吗？**
A：若因不符合录用条件被解除，公司无需支付经济补偿，但须有客观依据。其他情形按《劳动合同法》执行。

**Q6：离职证明多久能拿到？**
A：完成交接和离职手续后即可开具，一般在离职当天或手续办结时领取。
