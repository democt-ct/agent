# 考勤政策

**版本：** v2.0 | **生效日期：** 2024年3月1日 | **归口部门：** 人力资源部  
**最新修订日期：** 2026年5月19日

---

## 1. 定义与适用范围

### 1.1 关键定义
- **全勤员工**：在月度考勤周期（当月1日00:00至月末最后一天23:59）内，无任何**迟到**、**早退**、**旷工**、**漏打卡**且请假≤0.5天（经审批的病假、婚假、产假等法定带薪假除外）的员工。全勤员工将获得**200元/月**的全勤奖励，计入当月工资。
- **迟到**：未在规定的**核心到岗时间**或**标准上班时间**之前到达工作岗位且未提前报备。具体判定时间见第2节与第4节。
- **早退**：未到规定的**下班时间**未经审批擅自离开工作岗位。早退时长从规定下班时间前实际离岗时间开始计算。
- **旷工**：指以下任意一种情况：
    - 未打卡且未提交任何请假/补卡申请。
    - 单次迟到/早退时长超过**60分钟**且未获批。
    - 连续三个工作日未打卡且无联系记录，视为严重违纪。
- **加班**：经审批的、在标准工时之外的持续工作，以**0.5小时**为最小计算单位。不足0.5小时不计。加班时长以打卡记录为准，若下班打卡后未立即开始加班，需在加班申请中备注实际开始时间。
- **调休**：将加班时长兑换为等时的带薪休假，调休有效期自加班发生日起**90个自然日**内有效，逾期自动失效并按加班费计算。

### 1.2 适用范围
本政策适用于公司**全体正式员工**（含试用期、实习生）。外包人员、小时工、劳务派遣人员参照执行，具体以合同约定为准。**不定时工作制岗位**（如高级管理人员、外勤销售经理、部分研发岗）不适用本考勤政策中的打卡与工时规定，但须遵守周出勤天数、会议到岗及项目交付要求，并在OA系统中按月提交工作报告。

## 2. 工作时间

### 2.1 标准工时制
- **上班时间**：周一至周五 **08:30**（核心到岗时间 08:30~09:00，超出视为迟到）。
- **下班时间**：周一至周四 **18:00**，周五 **17:30**。
- **午休时间**：**12:00 ~ 13:00**（固定1小时，不计入工作时间，不扣减餐补）。此段时间为员工自由支配时间，可离开公司、用餐或休息。
- **休息日**：周六、周日为双休日。若遇法定节假日调休，以国务院通知及公司通知为准。

### 2.2 弹性工时制
适用于以下经审批的岗位：**销售部、运维部、客服部、研发部（部分项目制工程师）**。执行规则如下：
- **核心到岗时间**：每日 **10:00 ~ 16:00** 必须在岗（含午休1小时，但核心时段内不可离岗超过30分钟）。若员工在此时间段内离岗超过30分钟，需在钉钉申请“临时外出”。
- **日工作时长要求**：全天在岗时长不少于 **6小时**（不含午休），周总工时不少于 **30小时**。
- **考勤记录**：弹性工时员工同样需打卡（刷脸或钉钉），但迟到判定按核心到岗时间（10:00）执行。

#### 标准工时 vs 弹性工时对比

| 对比项               | 标准工时制                 | 弹性工时制                     |
|----------------------|----------------------------|--------------------------------|
| 上班时间             | 08:30                      | 灵活，核心时段 10:00~16:00    |
| 下班时间             | 18:00(周一~四)/17:30(周五) | 灵活，满足日工时≥6h即可       |
| 午休时间             | 12:00~13:00 固定           | 可灵活安排，但核心时段不可离岗 |
| 适用岗位             | 行政、财务、人事等后台部门 | 销售、运维、客服、部分研发     |
| 签到要求             | 每日两次（上班+下班）      | 每日两次，可外勤打卡           |
| 迟到判定标准         | 超过08:30                  | 超过10:00                     |

## 3. 打卡规则

### 3.1 打卡方式与有效时段
- **公司大门刷脸**：有效时段 **07:30 ~ 09:30**（上班打卡）及 **17:00 ~ 19:00**（下班打卡）；超出该范围需使用钉钉外勤打卡。刷脸失败可重复尝试，若连续3次失败，请使用钉钉打卡。
- **钉钉APP外勤打卡**：适用于出差、外勤、通勤异常等场景。需在 **离开公司前**或 **到达客户现场时** 开启GPS定位拍照打卡。若无法定位，可手动输入地址+拍照。
- **补卡申请**：因网络故障、设备故障、系统错误导致无法打卡，或忘带手机/工牌，可在 **当日24:00前** 提交补卡申请，附证人证明或现场照片。

### 3.2 打卡异常处理流程
| 异常类型             | 判定标准与处理方式                                                                 | 扣款/影响                        |
|----------------------|------------------------------------------------------------------------------------|----------------------------------|
| 忘打卡（已到岗）    | 需当日提交补卡申请，附证人（同级/上级）证明+工作轨迹截图（如邮件发送记录、会议签到） | 不扣款，但月累计≥3次需HRBP谈话 |
| 迟到（>08:30）       | 分时段扣款（见第4节）                                                             | 按标准扣款                       |
| 早退（<17:30离岗）   | 早退≤15分钟：提醒警告；15~30分钟：扣半日餐补；>30分钟：按旷工半天处理            | 扣款+旷工记录                    |
| 忘带工牌             | 在保安处登记身份证+人脸核验后可进入，不扣款                                       | 无影响                           |
| 外勤打卡未开GPS      | 视为无效打卡，需在24小时内补交位置证明                                             | 按缺勤处理直至补交               |

### 3.3 一个月内打卡异常累计规则
- 月累计 **迟到次数超过3次**，从第4次起每次按旷工半天处理，即扣半日基本工资+无餐补。
- 月累计 **漏打卡次数超过5次**，从第6次起每次扣款50元，直接从当月工资中扣除。

## 4. 迟到与早退扣款标准

| 迟到/早退时长 | 扣款金额                                      |
|---------------|-----------------------------------------------|
| 1 ~ 15 分钟   | 扣除当日餐补（20元）                          |
| 16 ~ 30 分钟  | 扣除当日餐补 + 30元（共50元）                 |
| 31 ~ 60 分钟  | 扣除当日餐补 + 60元（共80元）                 |
| 超过 60 分钟  | 视为旷工半天（扣除半日基本工资+无餐补）         |

> **注**：餐补为每日20元，仅在全勤且正常在岗≥4小时时发放。扣款从当月薪资中扣除。

**示例**：员工A周一08:50到岗，迟到20分钟，当月已迟到2次。该次扣款为：餐补20元+30元=50元。若当月第4次迟到（即使仅5分钟），则按旷工半天处理，扣款为半日基本工资（假设日薪200元，即扣100元）+无餐补，且影响全勤奖。

## 5. 外出与出差

### 5.1 临时外出（≤2小时）
- 直属上级**口头同意**后，在钉钉填写“临时外出登记”表单，注明目的地、预计返回时间。
- 外出期间仍需保持钉钉在线，返回后及时销假（系统自动）。
- 每月临时外出累计不得超过**5次**，超过须改为OA外出申请。

### 5.2 半天以上外出/出差
- 提前 **1个工作日** 提交OA“出差/外出申请”，填写起止时间、行程安排、预算（如需），经上级审批后生效。
- 出差结束后 **3个工作日内** 提交出差报告（出差≥3天需附成果清单）。
- 出差期间的考勤以外勤打卡记录为准，无需补卡。

### 5.3 外勤打卡规范
- 离开公司前在钉钉开启“外勤模式”，系统自动记录GPS坐标及时间。若无法定位，可手动输入地址+拍照。
- 外勤打卡记录与出差申请单匹配，如无对应申请，按缺勤处理。

## 6. 加班处理

### 6.1 加班申请与审批
- **加班必须提前申请**：工作日加班须在当日 **17:00前** 提交OA加班申请；休息日/节假日加班须在加班开始前 **1个工作日** 提交。
- **紧急加班**：可事后 **24小时内** 补申请，但需上级在申请中备注紧急原因。
- 审批流程：员工 → 直属上级（审批） → 部门负责人（知会） → 人力资源部（存档）。

### 6.2 加班时长计算
- 以 **0.5小时** 为最小单位，不足0.5小时不计。
- 加班时长以打卡记录为基准，若下班打卡后未立即开始加班，需在加班申请中备注实际开始时间。

### 6.3 加班补偿方式
| 加班类型       | 补偿方式                                      | 限制条件                        |
|----------------|-----------------------------------------------|---------------------------------|
| 工作日加班     | 调休（1:1）或加班费（时薪×1.5倍）           | 调休有效期90天，逾期转加班费    |
| 休息日加班     | 优先调休（1:1.5）或加班费（时薪×2.0倍）     | 调休须在加班后60天内休完        |
| 法定假日加班   | 加班费（时薪×3.0倍），不可调休替代           | 强制支付                         |

> **示例**：员工B月薪6000元，日薪=6000/21.75≈275.86元，时薪≈34.48元。工作日加班2小时，加班费=34.48×1.5×2=103.44元；若选择调休，可休2小时带薪假。

## 7. 考勤异常申诉

### 7.1 申诉流程
员工对考勤记录（迟到、缺卡、旷工等）有异议，可在**5个工作日内**（自考勤异常发生日起算）提交申诉：

1. 登录OA → 考勤管理 → 点击“考勤申诉” → 选择异常记录。
2. 上传证明材料（以下任一种即可）：
   - 打卡失败截图或设备故障照片
   - 打车/地铁记录（证明到岗时间）
   - 就医记录（请假或迟到因突发疾病）
   - 当日工作内部沟通记录（证明已到岗）
3. 审批人：部门HRBP，2个工作日内反馈。

### 7.2 FAQ
| 常见问题                                   | 解答                                                                 |
|--------------------------------------------|----------------------------------------------------------------------|
| 忘记打卡但确实到岗了，怎么处理？           | 24小时内提交补卡申请，附证人证明或工作记录。月累计≤3次不扣款。     |
| 出差期间需要每天打卡吗？                   | 需要，使用钉钉外勤打卡，出差申请单会自动关联。                      |
| 调休可以跨年使用吗？                       | 原则上须在加班发生后90天内休完，跨年以自然年计算，未休完自动转加班费。 |
| 迟到多长时间算旷工？                       | 单次迟到超过60分钟，或月迟到累计第4次起，按旷工半天处理。           |
| 弹性工时员工如何计算加班？                 | 超出核心时段（10:00~16:00）且经审批的工作时长，按加班规则计算。    |
