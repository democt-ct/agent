## 1. 适用范围与定义

### 1.1 适用范围
本办法适用于公司所有正式员工（含试用期员工）因工作需要申领公司标准办公设备的行为。具体涵盖以下资产类别：

- **核心计算设备**：台式电脑（含工作站）、笔记本电脑、瘦客户机
- **显示与输入设备**：显示器、键盘、鼠标、绘图板、手写笔
- **移动与通讯设备**：工作手机（仅限销售、运维、外勤岗位）、平板电脑（仅限演示、现场服务场景）
- **会议与外设**：网络摄像头、专业耳麦、会议扬声器、多功能扩展坞、投影仪（短时借用）
- **特殊设备**：经部门负责人与信息技术部共同确认的行业专用设备（如：便携式打印机、专用扫描仪等）

> **严格排除项**：
> - 员工个人自购设备（即使用于工作，也不适用本流程，维修、数据安全责任由个人承担）
> - 非标消费品（如：普通U盘、鼠标垫、手机支架等，请按办公用品申购流程办理）
> - 纯软件授权与SaaS账号（请走软件资产申请流程）

### 1.2 核心定义
- **标配设备**：依据本办法 2.1 条规定的岗位对应型号与配置的资产，公司全额承担费用，所有权归公司。
- **超配设备**：员工申请的设备型号或配置高于岗位标配标准的资产。超配部分差价由申请部门或员工个人承担。
- **备用设备**：信息技术部库存中用于应对设备突发故障或新员工短期过渡的临时周转设备。使用期限最长**30个自然日**，期满必须归还或发起正式申领流程。
- **资产生命周期**：从设备验收、入库、领用、使用、维修、调拨到最终报废或归还的完整过程，所有环节必须在IT资产管理系统（“系统”）中记录。

## 2. 申领条件与资格审核

### 2.1 岗位硬件配置标准（标配）

员工的设备配置依据其岗位属性与工作内容分为四个层级，且随公司年度采购计划动态调整。以下为当前有效标准：

| 岗位类型 | 核心设备（电脑） | 第二显示设备 | 其他标配外设 | 网速要求 | 适用场景说明 |
|---------|----------------|-------------|-------------|---------|-------------|
| **行政/职能岗** | 笔记本电脑（ThinkPad E14 Gen6 或同等） | 23.8英寸FHD显示器×1 | 标准键盘、有线鼠标、USB耳机 | ≥50Mbps | 日常办公、文档处理、非性能敏感应用 |
| **技术/研发岗** | 高性能笔记本（ThinkPad X1 Carbon Gen12 或 MacBook Pro 14" M4） | 27英寸4K显示器×1 | 全功能扩展坞、降噪耳机 | ≥100Mbps，优先有线网络 | 代码编译、虚拟化环境、持续集成、原型设计 |
| **管理岗（总监及以上）** | 轻薄型笔记本（MacBook Air 15" M3 或 ThinkPad X1 Nano） | 27英寸4K显示器×1 | 降噪耳机、无线鼠标 | ≥100Mbps | 高频会议、移动办公、演示汇报 |
| **设计/多媒体岗** | 高性能工作站（MacBook Pro 16" M4 Max 或 Dell Precision） | 27英寸专业校色显示器（99% sRGB）×1 | 标准手绘板、校色仪（按需）、全功能扩展坞 | ≥100Mbps，必须有线，建议专用带宽 | 图形设计、视频剪辑、3D建模、渲染 |

> **说明**：
> - 实习期员工（前6个月）及入职未满3个月的新员工，仅发放**备用设备**使用权，不触发标配新购流程。
> - 如因业务需要同时使用多台显示器或叠加外设，需在OA系统中填写“额外设备需求理由说明”，由直属上级与信息技术部共同审批。

### 2.2 申领资格条件

申请人在提交设备申领申请时，必须且至少满足以下**一项**条件，并提供对应证明文件：

1. **新员工入职满3个月**：从入职之日起计算，满90个自然日。需提供**转正审批通过通知**。
2. **原设备发生物理损坏且无法修复**：该设备需已在系统中有至少一次的**官方维修记录**（非个人原因，如液体泼溅、意外跌落等除外）。需上传**维修工单或检测报告截图**。若设备已过保且维修评估成本超过设备原值的50%，且非人为故意损坏（需填写《设备损毁责任认定表》），则视为“无法修复”。
3. **岗位调动或业务需求变更**：员工在系统内的岗位信息发生变更（需人力资源部提供**调动通知**），且新岗位的标配设备与旧设备不兼容（例如：原为行政岗调至设计岗，现有笔记本无法满足设计软件性能要求）。需由**调入部门**负责人签字确认。
4. **设备自然淘汰或超期服役**：设备已使用超过**4年**，且性能已无法支撑当前主流软件运行。需提交设备基准测试结果（由信息技术部免费提供），由资产管理专员确认后进入申领流程。

> **严禁情形**（出现任一即驳回申请）：
> - 在原设备无任何故障且满足工作需求的情况下，仅因个人偏好（如偏好不同品牌、颜色）申请更换。
> - 申请时存在未归还的旧设备、备用设备或外借设备。
> - 在申请前3个月内有过因人为损坏导致公司资产损失的记录。

## 3. 完整申领与审批流程

### 3.1 标准流程序列

启动申领时，必须遵循以下步骤，每步未完成即终止申请流程：

1. **在线提交申请**：
   - 登录OA系统 → “资产与行政” → “设备申领” → 填写《办公设备申领单》。
   - **必填项**：所在部门、岗位名称、申领设备类别（如“笔记本电脑”）、用途描述。
   - **附件必须上传**：如属于1.2条中任一条件，上传对应的证明文件（如：调动通知、维修工单、转正通知、部门负责人确认函）。
   - **特殊说明**：如申请的是**超配设备**，需在“特殊说明”栏写明理由和差价承担方案。

2. **直属上级审批**（1个工作日）：
   - 上级确认申请岗位与实际岗位一致，且该设备为完成本职工作必需。
   - 如上级审批通过，系统自动流转至信息技术部。

3. **信息技术部库存与预算审核**（1个工作日）：
   - 资产管理员在系统中检查对应类别库存情况。
     - **有库存**：直接跳转至第4步（发放流程）。库存设备必须是经过翻新或固件升级、符合当前岗位最低配置标准的备用机，使用年限不超过2年。
     - **无库存**：自动触发采购流程（见3.2条）。
   - **预算校验**：信息技术部核对部门年度IT设备预算是否有结余。若超预算，需转部门总监审批追加预算。

4. **采购与到货通知**：
   - 采购时长依据设备类型（见下3.2节）。
   - 到货后，信息技术部在系统更新库存，并向申请人发出**“设备已到货，请于5个工作日内领取”**的OA通知。超期7个工作日未领取，设备将重新归入仓库，申请单作废，需重新提交申请。

5. **领用与签收**（到货后3个工作日内必须完成）：
   - 员工携带**工牌与身份证**（原件）前往信息技术部办公点。
   - 由资产管理员进行设备激活、安装公司标准镜像系统和必要的安全软件（如：EDR、VPN客户端、公司统一Office、企业微信、加密软件）。
   - 双方共同核对硬件配置与附件（电源适配器、鼠标、键盘、保修卡等）。
   - **必须签署**《设备交付确认单》（一式三联：员工、IT、财务各一联）。签署即表明：
     - 员工已收到且后描述的设备完好；
     - 员工承诺遵守《公司资产管理制度》，对设备负有保管责任。
   - 资产管理员同时在IT资产系统进行**设备登记**，将设备编号与员工信息绑定，完成资产入库。

6. **财务与折旧处理**（后台自动发生）：
   - 信息技术部于每月25日统一将当月设备交付记录报财务部，启动设备折旧，并计入对应部门费用。

### 3.2 采购周期明细

| 设备类型 | 标准采购周期 | 加急采购周期（需总监审批） | 库存替代方案 |
|---------|------------|-------------------------|-------------|
| **标配型号笔记本电脑/台式机** | 1~3个工作日 | 当日（特殊调配）；24小时内 | 使用同性能段备用机 |
| **高性能工作站/专业显卡机型** | 7~10个工作日 | 3~5个工作日（需支付额外采购运费） | 无替代，必须等待 |
| **特殊配置/非标外设**（如专业手绘板、2米以上升降桌） | 10~15个工作日 | 5~7个工作日 | 借调部门同组同事设备（征得同意） |
| **批量采购（>5台同型号设备）** | 15~20个工作日（含集团采购合同周期） | 不支持加急 | 分批发货：首批≤2台，7日内 |

> **紧急申领定义**：因**生产环境故障、客户现场紧急演示、重大事故处理**导致当前设备完全无法使用，且无备用机可用。紧急申领需先电话联系信息技术部总监（分机8001），获得口头同意后，再于24小时内补全OA流程与审批附件。

### 3.3 常见常见问题解答（FAQ）

**Q1：入职刚满2个月，我需要一台专用电脑，能申请吗？**
A：不能。根据1.2条，新员工前3个月（试用期内）只能使用备用设备。请向直属上级或部门行政申请备用机领取。建议您先使用公共区域工位电脑或借用组内备用机过渡。

**Q2：我离职前，设备需要归还到什么地方？**
A：请参考本管理办法第5.1条。必须由本人携带设备、电源、所有附件到信息技术部前台进行数据清除和硬件验收。**不可**委托他人代还，**不可**直接交给部门同事或打包装箱寄回。

**Q3：我申请的显示器到货后，发现尺寸太大，桌面放不下，能换成小尺寸的么（标配）？**
A：可以。请在到货签收后**5个工作日内**联系信息技术部。信息技术部将进行现场勘查，如确因物理空间限制（测量桌面实际深度＜60cm等），且您愿意承担一次换货的物流手续费（50元，从个人薪资中扣除），信息技术部可为您调换为23.8英寸显示器。若超过5个工作日未反馈，视作签收接受，后续更换按“超配申请”处理，差价由个人承担。

**Q4：自购设备用于工作，能否享受公司IT部门的软件或网络支持？**
A：不能。自购设备不属于企业资产，信息技术部不为其提供软件安装、网络配置或安全补丁服务，也不对其数据安全负责。某些业务应用（如VPN客户端）可能无法在非管理设备上安装。建议您尽快提交正式设备申领，使用公司标准环境。

**Q5：公司能提供苹果电脑的Windows系统镜像吗？**
A：可以。对于批准配备MacBook的员工，信息技术部默认提供macOS原生原厂系统。若您需要运行Windows虚拟机（例如必须使用某些Windows-only的行业软件），信息技术部可以为您**预安装Parallels Desktop + Windows 11 企业版授权**，在您签收时一次性完成。请确保硬盘容量足够（建议至少256GB空闲空间）。

## 4. 费用承担与结算规则

| 申领场景 | 费用承担方 | 特殊说明 |
|---------|----------|---------|
| **岗位标配内设备**（按2.1条） | 公司（计入部门IT预算） | 无 |
| **员工申请超配设备**（如：行政岗硬要M4 Max工作站） | 员工个人承担差价（直接从当月工资扣除） | 单次限差价≤3000元。超出需分月扣（最长12个月）。申请人需在OA填写《个人费用承担确认书》并签字。若员工在公司服务不满1年离职，剩余未扣清差价一次性从离职结算中扣除。 |
| **部门为特定项目申请超配设备** | 部门经费（需部门总监特批） | 需在OA附件上传《项目预算调整说明》，由财务部审核。 |
| **人为损坏导致的维修或更换** | 责任人个人承担（维修费或设备折旧残值） | 需要填写《资产损毁责任认定表》并由信息技术部进行定责评估。若确属恶意损坏（如：暴力敲击、故意液体泼溅），公司保留追究其纪律处分或法律责任的权力。 |
| **自然损耗或因设计缺陷导致的设备故障** | 公司（由原厂保修覆盖或公司固定维修渠道承担） | 需提供信息技术部出具的《非人为故障确认函》作为证明。 |

## 5. 设备归还、调动与报废管理

### 5.1 离职归还流程（必须严格执行）

所有离职员工（无论是主动离职、被动离职、退休或合同到期不续签）必须遵循以下时间线和操作：

1. **提前预约**：在最后工作日前**至少3个工作日**，在OA系统提交“离职设备归还申请”，预约信息技术部资产管理员的时间。
2. **携带物品**：工牌（用于核验身份，归还后由信息技术部当场回收并剪角作废）+ 全部申领的设备及所有包装/附件/配件（尤其要包括电源适配器、线缆、触控笔、遥控器、保修卡盒、原装包装箱）。
3. **现场操作**：
   - 信息技术部检查设备外观（外壳、屏幕、键盘、接口）、功能（开机、网络、摄像头、扬声器）、与后台资产系统信息是否一致。
   - 设置设备进入**企业级深度恢复模式**，彻底删除员工所有个人数据与应用（不可恢复级别。此过程不备份员工个人数据，请员工提前完成数据迁移和归档）。
   - 双方签署《设备归还证明》（一式两份：员工、IT各一份），员工将此证明扫描件交人力资源部备案，否则人力可暂缓发放工资或办理离职手续。
4. **状态确认**：信息技术部在后台上传归还记录的2个工作日内，更新系统资产状态为“库存-已归还”，并释放该员工的设备占用。

### 5.2 内部调动流程

- 当员工发生跨部门或同部门内部岗位调动时，其**已配发的设备（含外设）原则上随人转移**。员工需在调动生效后的7个工作日内，在OA系统中提交“设备关联岗位变更申请”（附人力资源部调令）。
- **调入部门如需追加设备**：按本管理办法第2.1条与3.1条新员工流程重新申请。注意：原有的调动带来的设备不算“新员工入职”，因此触发的是“岗位变更+追加设备”流程。
- **调出部门无需单独审批**设备转移事宜，但信息技术部会在后台进行资产归属的自动划转。若设备型号不符合新岗位标准（如调出是设计岗，新岗位是行政岗，设备从MacBook Pro降级为ThinkPad），则：
  - 调入部门需退还高端设备（调入信息技术部仓库），并为员工申领新岗位标配设备。
  - 若个人恰好希望保留高端设备（超配），则需按超配申请流程处理差价。若部门愿意承担保费，由部门总监审批签字。

### 5.3 设备报废管理

- 当设备服役时间超过5年，且信息技术部评估维修成本超过设备残值80%或已无原厂维修支持，将被系统自动标记为“报废待审”。
- 信息技术部每季度统一审核，对达标设备进行**数据销毁（物理或软件级，符合ISO 27001标准）** → **部件拆解回收（如内存、硬盘可循环利用）** → **无害化处理（电子垃圾回收公司）**。
- 员工个人不得擅自处理任何报废状态的设备，否则视为偷盗公司资产。
