# 远程办公IT指南

**版本：** v1.0 | **生效日期：** 2026年6月1日 | **归口部门：** 信息技术部

---

## 1. 适用范围

本指南适用于经批准的居家办公、出差期间远程接入公司系统的员工。远程办公须经直属上级审批，并遵守本指南的安全规范。

---

## 2. 远程接入要求

### 2.1 必须使用 VPN

- 访问公司内网系统（OA、ERP、代码仓库、内部文档）**必须通过公司 VPN**。
- 禁止在公共 WiFi 下不开 VPN 直接访问公司系统。
- VPN 客户端安装与配置见《网络和VPN配置》。

### 2.2 设备要求

| 要求 | 说明 |
|------|------|
| 使用公司设备 | 优先使用公司配发设备，个人设备需经 IT 审批并安装安全软件 |
| 系统更新 | 操作系统和安全补丁保持最新 |
| 杀毒软件 | 公司统一部署的 EDR 必须运行 |
| 屏幕锁定 | 离开设备时立即锁屏，设置5分钟自动锁屏 |
| 磁盘加密 | 公司设备默认开启 BitLocker/FileVault |

### 2.3 网络环境

- 优先使用家庭宽带，避免使用公共 WiFi（咖啡厅、酒店、机场）。
- 如必须使用公共 WiFi，**必须先连接 VPN** 再访问任何公司资源。
- 手机热点是公共 WiFi 的安全替代方案。

## 3. 远程协作工具

| 工具 | 用途 | 访问方式 |
|------|------|---------|
| 企业微信 | 即时通讯、视频会议 | 直接访问，无需 VPN |
| OA 系统 | 审批、考勤、报销 | 需 VPN |
| 代码仓库 | 代码提交与审查 | 需 VPN |
| OneDrive | 文件共享与协作 | 直接访问，无需 VPN |
| 邮件（Outlook） | 邮件收发 | 直接访问，无需 VPN |

---

## 4. 安全注意事项

- **不在公共场所讨论保密信息**（电话、视频会议注意周围环境）。
- **不将工作文件下载到个人设备**，使用网络盘在线编辑。
- **不使用个人邮箱、微信传输工作文件**。
- **发现异常立即上报**：设备丢失、账号异常、可疑邮件，第一时间联系 IT。

---

## 5. 常见问题（FAQ）

**Q1：VPN 连不上怎么办？**
A：参考《网络和VPN配置》自排查步骤。仍无法解决，联系 IT 服务台（工单或热线）。

**Q2：个人电脑可以用来远程办公吗？**
A：需经 IT 审批，安装公司 EDR 和 VPN 客户端后方可使用。

**Q3：远程办公期间考勤怎么打？**
A：使用钉钉外勤打卡，开启 GPS 定位，详见《考勤政策》外勤打卡规范。

**Q4：视频会议背景有要求吗？**
A：建议使用虚拟背景或整洁环境，避免暴露家庭隐私或公司保密信息。
